# Business impact

The planning output is meant to help with three questions.

- Which links are likely to cross the congestion threshold soon?
- Where should capacity work be scheduled first?
- Which links can wait until the next change window?

The forecast is most useful when it is translated into a short planning note that a manager or service owner can read quickly. A link that stays above 80 percent utilization for several days is usually a better upgrade candidate than a link with a one-day spike.
